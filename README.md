# alg_lab_2_zad_3


**ФОРМУЛИРОВКА**


Это 3 задание лаборатоной работы №2. Приложение написано на языке C# и с помощью Windows Forms реализует графический интерфейс для общения с пользователем. Оно принимает на вход число от пользователя (не более 20 цифр в числе) и строит с помощью PictureBox гистограмму для всех цифр в введённом числе. Также, как и в предыдущих заданиях лаборатоной работы №2, в приложении присутствует кнопка помощи и формулировки задания.


**КАК ЗАПУСКАТЬ**


Чтобы запустить это приложение, скачайте даннный репозиторий и запустите sln файл. Если всё прошло успешно - пользуйтесь рабочей формой в соответствии с инструкциеми (для этого в приложении реализована кнопка с помощью). В случае возникновения ошибок (такое увы бывает) просто перейдите и закомментируйте ошибочные 8 строк. Готово! От этого приложение не теряет своих способностей.
